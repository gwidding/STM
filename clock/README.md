# STM32F429를 활용한 디지털 알람시계 제작

수행기간 : 2023.11.10 ~ 2023. 11.21  (약 10일)
업무 : 회로제작, STM32 펌웨어 개발  
사용기능 : Timer, GPIO, EXT_INT, ADC(with DMA), Flash 메모리, PWM  
사용 디바이스 : Buzzer, 조이스틱, 16x2 character LCD  
사용 툴 : STM32CubeIDE  
사용 보드 : NUCLEO-F429ZI
